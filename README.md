# Arcanaz

Knowledge Beyond Definitions. The Modern Knowledge Platform.

Arcanaz is a premium knowledge platform that blends dictionary, encyclopedia, and AI learning assistant into a single, elegant interface reminiscent of an ancient library reimagined for the AI era.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
